# ‚ö° Electricity Billing System

## üöÄ Features

### **User Features**
- Calculate monthly electricity bills  
- View past bills  
- OTP-based phone verification (AWS SNS)  
- Receive email notifications via SMTP  

### **Admin Features**
- Manage customers (Add / Update / Delete)  
- Update electricity unit pricing table  
- View all bills and system logs  
- Role-based access (Admin vs User)
